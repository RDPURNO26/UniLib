## Getting Started

1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development server.

## Development Process

This project was built through a collaborative "pair programming" workflow:
- **Human Developer**: Directed the project vision, debugged complex issues, refined architectural decisions, and finalized logic.
- **AI Assistant**: Provided code generation and rapid prototyping support under human supervision.
